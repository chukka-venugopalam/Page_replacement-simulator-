Page Replacement Simulator & Analysis

Concepts & Time Complexity

FIFO (First-In, First-Out)

Concept: Replaces the oldest page in memory.

GATE Note: Suffers from Belady's Anomaly (increasing frames can increase page faults). Check the line graph below to see it!

Time Complexity: O(1) per step.

LRU (Least Recently Used)

Concept: Replaces the page not used for the longest time.

GATE Note: Stack algorithm; guaranteed to never suffer from Belady's Anomaly.

Time Complexity: O(N) per step (array approach).

OPT (Optimal)

Concept: Replaces the page that will not be used for the longest time in the future.

GATE Note: Yields the theoretical minimum faults. Impossible to implement practically.

Time Complexity: O(N) per step.
